# PlayerStatsMenu - 玩家状态信息菜单插件

一个用于查看玩家血量、护甲、移动速度等状态信息的Minecraft服务器插件。

## 功能特性

- 📊 **全面的状态信息显示**：
  - 血量信息（当前/最大生命值）
  - 护甲值（穿戴装备防护）
  - 移动速度（行走/飞行速度）
  - 饥饿值（饥饿等级/饱和度）
  - 经验等级（等级/经验进度）
  - 游戏模式（当前模式/状态）
  - 在线时间（累计游戏时间）
  - 当前世界（世界信息/环境）

- 🎮 **多种打开方式**：
  - 命令：`/stats`
  - 物品右键：状态查看器
  - 管理员可为其他玩家打开

- 🎨 **美观的GUI界面**：
  - 4行自定义GUI布局
  - 彩色物品图标
  - 详细的信息描述
  - 点击查看详细信息

- ⚙️ **高度可配置**：
  - 所有物品可自定义
  - 消息可本地化
  - 功能开关控制
  - 权限节点管理

## 安装方法

1. 将编译后的 `.jar` 文件放入服务器的 `plugins` 文件夹
2. 重启服务器或使用 `/reload` 命令
3. 插件会自动生成配置文件

## 使用方法

### 基础命令
- `/stats` - 打开自己的状态信息菜单
- `/stats help` - 查看帮助信息

### 管理员命令
- `/stats open <玩家>` - 为指定玩家打开菜单
- `/stats give [玩家]` - 给予状态查看器物品
- `/stats reload` - 重新加载配置文件

### 物品使用
1. 使用 `/stats give` 获取状态查看器
2. 手持物品右键即可打开菜单
3. 菜单中点击物品查看详细信息
4. 点击刷新按钮更新数据

## 权限节点

- `playerstatsmenu.use` - 使用菜单和物品（默认所有玩家）
- `playerstatsmenu.admin` - 管理员功能（默认OP）

## 配置文件

配置文件位于 `plugins/PlayerStatsMenu/config.yml`，主要配置项：

### GUI设置
```yaml
gui:
  title: "&6玩家状态信息"  # GUI标题
  rows: 4                 # GUI行数
```

### 物品设置
```yaml
items:
  health:
    material: "RED_STAINED_GLASS_PANE"
    displayname: "&c❤ 血量"
```

### 功能开关
```yaml
features:
  command:
    enabled: true    # 是否启用命令
  item:
    enabled: true    # 是否启用物品
```
